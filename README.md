# YODECO - Youth Democratic Coalition

A premium, modern NGO website for **YODECO**, a youth-driven political and civic movement in Nigeria.

## Features
- **Modern Design**: High-end aesthetics with glassmorphism and smooth animations.
- **Fully Responsive**: Optimized for desktop and mobile devices.
- **Interactive Donation Flow**: Multi-step donation process with bank selection.
- **Dedicated Contact Page**: Functional inquiry form with instant feedback.
- **SEO Optimized**: Semantic HTML and performance-focused asset management.

## Tech Stack
- HTML5
- CSS3 (Grid/Flexbox/Animations)
- Vanilla JavaScript (ES6+)
- FontAwesome Icons
- Google Fonts (Outfit & Inter)

## Setup
Simply open `index.html` in any modern web browser to view the site locally.
